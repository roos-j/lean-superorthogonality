### Formalization of type IV superorthogonality

This is an autoformalization of the main result (Theorem 1) in the paper

P. T. Gressman, L. B. Pierce, J. Roos, and P.-L. Yung. [A new type of superorthogonality.](https://arxiv.org/abs/2212.08956) *Proc. Amer. Math. Soc.* 152, 665-675, 2024. 

The bulk of the ~3k lines of code in this project was generated using Codex/gpt-5.5-xhigh. Specifically, the files in the `Codex` folder contain the autoformalized proofs while the remaining files contain the basic definitions and main theorem statements, which were human generated.
